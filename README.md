# Nexa

> Your Intelligent AI Workspace

Nexa is a production-ready AI chatbot platform inspired by modern AI assistants. It supports multiple AI providers, streaming responses, RAG, document chat, image understanding, model comparison, and more.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- FastAPI
- Supabase
- Gemini
- Ollama
- Docker
- Turborepo
